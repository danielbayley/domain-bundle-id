Domain ⮂ _[Bundle ID]_
=======================
Generate a macOS _[bundle identifier][bundle id]_ from a [URL]/[domain].

Install
-------
~~~ sh
npm install @danielbayley/domain-bundle-id #--save-dev
~~~

Contribute
----------
~~~ sh
git config core.hooksPath .github/hooks
~~~
